1. Konstruktion von gleichseitigen Dreiecken¶

in der Ebene und in der $xy$-Ebene des Raumes

Ausgehend von zwei Punkten $A$ und $B$ wird die Lage des dritten Punktes $C$ auf der
Mittelsenkrechten der Grundseite berechnet

Zur Anwendung in der $xy$ - Ebene des Raumes sind folgende nderungen zu machen:

  • ebenfalls nicht definiert ist die $koord$-Eigenschaft fr Geraden, wohl aber die $prg$-

Berechnungen im allgemeinen Fall¶

Mit den angefhrten nderungen wird ein im Sinne der Aufgabenstellung
allgemeingltiger Algorithmus erarbeitet. Dieser wird als Python-Funktion
bereitgestellt, die das erzeugte Dreieck zurckgibt

In [11]:
# Definition der Funktion

# Parameter sind die Endpunkte der Ausgangsstrecke

# Es wird die dim-Eigenschaft (Dimension) benutzt, die alle 
# agla-Objekte haben

def gleich_seitiges_dreieck(A, B):
    
    c = Strecke(A, B) 
    M = c.mitte
    
    ri = c.gerade.richt   # Richtungsvektor der Tgergeraden von c
    if A.dim == 2:        # Unterscheidung R^2 und R^3
        norm = Vektor(-ri.y, ri.x)   # Normalenvektor der Strecke
    else:
        norm = Vektor(-ri.y, ri.x, 0)
         
    ms = Gerade(M, norm)
    C = ms.pkt(t)
    
    L = lse(Abstand(A, C) - c.lnge)   
    t0 = L[1][t]                         

    C = ms.pkt(t0)
    
    return Dreieck(A, B, C)

Fr den eventuell beabsichtigten allgemeinen Gebrauch der Funktion fehlen nur
noch Eingabekontrollen bezglich der Argumente und die Einbindung der Funktion in
die Arbeitsumgebung des Anwenders
(deutsche Umlaute und vernderte Operatoren sind dann nicht mehr zu verwenden)

Man kann sich berlegen, dass bei rationalen Koordinaten von $A$ und $B$ mindestens
eine der Koordinaten von $C$ eine irrationale Zahl (Wurzelausdruck) ist. Somit besteht
schon bei dieser einfachen Aufgabe die Notwendigkeit, mit irrationalen Koordinaten
von Punkten bzw. Komponenten von Vektoren rechnen zu knnen
